TTDeck 隱私政策

生效日期：2026-06-30

TTDeck 是唯讀車輛監控 app。它不是 Tesla 官方產品，不控制車輛，並且設計上連接使用者自行設定的 Companion Server。

1. 我們是誰

TTDeck 由 范晨涛（Fan Chentao） 提供。如需聯絡，請寄送電子郵件至 xxxkkk0101@foxmail.com。

2. 摘要

目前版本的 TTDeck 不包含廣告 SDK、第三方分析 SDK 或跨 app 追蹤。app 會在本機儲存連線設定，並使用唯讀權杖向你設定的 Companion Server 請求車輛摘要。車輛資料預期保留在你的 TeslaMate、你的 Companion Server、你的 iPhone，以及你自行選擇和控制的基礎設施中。

3. app 處理的資料

資料用途儲存或傳送位置
Companion Server 位址、資料來源模式、目前車輛、通知偏好、隱私顯示偏好設定 app、選擇顯示內容、記住本機偏好儲存在本機 iOS app 儲存空間中；開啟 iCloud 設定同步時，也會透過 Apple iCloud 鍵值儲存低頻同步到你的 iCloud 帳號
唯讀 Companion 權杖和裝置密鑰讓本機 iPhone 向你的 Companion Server 驗證儲存在 iOS Keychain 中，並只傳送給你設定的 Companion Server 位址
設定密鑰和一次性配對權杖建立或完成與 Companion Server 的裝置綁定設定密鑰由你輸入，並傳送給你設定的 Companion Server 執行設定操作。TTDeck 不會有意在設定操作後儲存它。
車輛狀態、電池、充電、安全、行程、路線、位置、遙測和診斷摘要顯示唯讀監控頁面、路線地圖、歷史記錄和本機通知檢查從你設定的 Companion Server 請求，並在本機顯示。app 設計上不會把這些資料傳送到開發者營運的雲端服務。
掃描配對 QR code 時的相機畫面掃描 Companion Server 配對 QR code僅在本機用於 QR code 識別。TTDeck 不儲存相機畫面。
你主動傳送給我們的支援郵件、截圖、日誌或診斷資訊回覆支援請求和排查問題僅在你主動提供時傳送。傳送前請移除權杖、精確住家位置、VIN 和其他敏感資訊。

4. Companion Server 和自託管資料

TTDeck 依賴你設定的 Companion Server。該伺服器可能讀取 TeslaMate、Postgres、MQTT、Tesla 服務、DNS 服務商和你選擇的託管服務。除非我們另行提供並明確標示開發者營運的託管服務，否則這些服務不由我們控制。你使用這些服務時也受其各自條款和隱私政策約束。

5. 第三方

TTDeck 使用 Apple 平台服務，例如 App Store、iOS Keychain、本機通知、MapKit 顯示、相機權限提示和用於低頻設定同步的 iCloud 鍵值儲存。Apple 透過 App Store 或 iOS 獨立收集的資料不由我們負責。如果本政策託管在 GitHub Pages，GitHub 可能為了安全和網站運作處理訪客 IP 位址等資訊。

6. 追蹤、廣告和出售資料

TTDeck 不出售使用者資料。目前 app 不使用第三方廣告 SDK、跨 app 追蹤或第三方分析 SDK。如果未來加入分析、崩潰報告、遠端推播、託管 Companion 服務或廣告，我們會在發布該版本前更新本政策和 App Store 隱私資訊。

7. 保留和刪除

本機設定會保留在裝置上，直到你修改、重置綁定或刪除 app。app 內的重置操作會清除本機唯讀權杖和裝置密鑰。撤銷伺服器端裝置存取、刪除 TeslaMate 資料、刪除 Companion Server 日誌和刪除託管基礎設施資料，需要在你控制的伺服器或服務中完成。

支援郵件會在回覆請求、保留記錄或履行法律義務所需的合理期限內保留。如需請求刪除你傳送給我們的支援資料，請聯絡 xxxkkk0101@foxmail.com。

8. 你的選擇

  • 使用展示模式，不連接 Companion Server。
  • 在 app 中修改或移除 Companion Server 位址和權杖。
  • 使用 app 的隱私顯示選項，在螢幕上隱藏精確位置、路線地圖、VIN 後綴、地點標籤和裝置權杖雜湊。
  • 在 TTDeck 或 iOS 系統設定中關閉本機通知。
  • 在 TTDeck 設定中重置本機綁定。
  • 在 Companion Server 支援的情況下，從伺服器端撤銷裝置存取。

9. 兒童

TTDeck 不面向兒童，也不應由兒童用於監控車輛或帳號資料。

10. 安全

TTDeck 使用 iOS Keychain 保存唯讀權杖和裝置密鑰。你需要自行保護 Companion Server、TeslaMate、資料庫、設定密鑰、託管帳號和網路設定。不要在公開日誌或支援渠道中分享原始權杖、設定密鑰、Tesla token、資料庫密碼、VIN、精確住家座標或完整路線資料。

11. 國際使用

你需要負責按適用於你和所在地的法律運行 Companion Server 和相關服務。

12. 變更

當 app、資料實務或法律要求變化時，我們可能更新本政策。更新後的政策會發布在本 URL，並標示新的生效日期。

13. 聯絡方式

隱私問題、隱私請求和支援請求可傳送至 xxxkkk0101@foxmail.com。
