Política de privacidad de TTDeck

Fecha de entrada en vigor: 2026-06-30

TTDeck es una app de monitorización de vehículos de solo lectura. No es un producto oficial de Tesla, no controla vehículos y está diseñada para conectarse a un Companion Server configurado por el usuario.

1. Quiénes somos

TTDeck es proporcionada por Fan Chentao. Puedes contactar con nosotros en xxxkkk0101@foxmail.com.

2. Resumen

La versión actual de TTDeck no incluye SDK publicitarios, SDK de análisis de terceros ni seguimiento entre apps. La app guarda ajustes de conexión en tu dispositivo y usa un token de solo lectura para solicitar resúmenes del vehículo al Companion Server que configures. Los datos del vehículo están pensados para permanecer en tu TeslaMate, tu Companion Server, tu iPhone y la infraestructura que elijas operar.

4. Companion Server y datos autoalojados

TTDeck depende de un Companion Server configurado por ti. Ese servidor puede leer TeslaMate, Postgres, MQTT, servicios de Tesla, proveedores DNS y proveedores de alojamiento elegidos por ti. No controlamos esos servicios salvo que proporcionemos e identifiquemos por separado un servicio alojado operado por el desarrollador. Tu uso de esos servicios se rige por sus propios términos y políticas.

5. Terceros

TTDeck usa servicios de la plataforma Apple como App Store, iOS Keychain, notificaciones locales, visualización MapKit, permisos de cámara e iCloud Key-Value Storage para sincronización de ajustes de baja frecuencia. No somos responsables de los datos que Apple recopile de forma independiente a través de App Store o iOS. Si esta política se aloja en GitHub Pages, GitHub puede tratar información de visitantes, como direcciones IP, para seguridad y operación del sitio.

6. Seguimiento, publicidad y venta de datos

TTDeck no vende datos de usuarios. La app actual no usa SDK publicitarios de terceros, seguimiento entre apps ni SDK de análisis de terceros. Si en el futuro añadimos análisis, informes de fallos, notificaciones push remotas, un servicio Companion alojado o publicidad, actualizaremos esta política y la información de privacidad de App Store antes de lanzar esa versión.

7. Conservación y eliminación

Los ajustes locales permanecen en tu dispositivo hasta que los cambies, restablezcas el enlace o elimines la app. La acción de restablecimiento de la app elimina de este dispositivo el token de solo lectura y el secreto del dispositivo. La revocación de acceso del lado servidor, la eliminación de datos de TeslaMate, registros del Companion Server o datos de infraestructura alojada deben realizarse en el servidor o servicio que controlas.

Los mensajes de soporte se conservan solo durante el tiempo razonablemente necesario para responder, mantener registros o cumplir obligaciones legales. Para solicitar su eliminación, contacta con xxxkkk0101@foxmail.com.

8. Tus opciones

  • Usar el modo demo en lugar de conectar un Companion Server.
  • Cambiar o eliminar en la app la URL del Companion Server y el token.
  • Usar las opciones de privacidad para ocultar ubicación exacta, mapas de ruta, sufijo VIN, etiquetas de lugares y hashes de tokens de dispositivo.
  • Desactivar las notificaciones locales en TTDeck o en Ajustes de iOS.
  • Restablecer el enlace de este dispositivo desde Ajustes de TTDeck.
  • Revocar el acceso del dispositivo en tu Companion Server, cuando sea compatible.

9. Menores

TTDeck no está dirigida a menores y no debe ser usada por menores para monitorizar datos de vehículos o cuentas.

10. Seguridad

TTDeck usa iOS Keychain para el token de solo lectura y el secreto del dispositivo. Eres responsable de proteger tu Companion Server, TeslaMate, base de datos, secreto de configuración, cuenta de alojamiento y configuración de red. No compartas tokens sin procesar, secretos de configuración, tokens de Tesla, contraseñas de base de datos, VIN, coordenadas exactas de casa ni rutas completas en registros públicos o canales de soporte.

11. Uso internacional

Eres responsable de operar tu Companion Server y servicios conectados conforme a las leyes aplicables a ti y a tu ubicación.

12. Cambios

Podemos actualizar esta política si cambian la app, las prácticas de datos o los requisitos legales. La versión actualizada se publicará en esta URL con una nueva fecha de entrada en vigor.

13. Contacto
