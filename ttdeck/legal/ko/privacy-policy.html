TTDeck 개인정보 처리방침

시행일: 2026-06-30

TTDeck은 읽기 전용 차량 모니터링 앱입니다. Tesla 공식 제품이 아니며 차량을 제어하지 않고, 사용자가 설정한 Companion Server에 연결되도록 설계되었습니다.

1. 제공자

TTDeck은 Fan Chentao가 제공합니다. 문의는 xxxkkk0101@foxmail.com로 보내 주세요.

2. 요약

현재 TTDeck 앱에는 광고 SDK, 제3자 분석 SDK 또는 앱 간 추적 SDK가 포함되어 있지 않습니다. 앱은 연결 설정을 기기에 저장하고 읽기 전용 토큰을 사용하여 사용자가 설정한 Companion Server에서 차량 요약을 요청합니다. 차량 데이터는 사용자의 TeslaMate, Companion Server, iPhone 및 사용자가 선택하고 관리하는 인프라에 남도록 설계되어 있습니다.

3. 앱이 처리하는 데이터

데이터목적저장 또는 전송 위치
Companion Server URL, 데이터 소스 모드, 선택한 차량, 알림 설정, 개인정보 표시 설정앱 설정, 표시 내용 선택, 로컬 설정 저장iOS 앱 저장소에 저장됩니다. iCloud 설정 동기화를 켜면 Apple iCloud Key-Value Storage를 통해 사용자의 iCloud 계정에도 저빈도로 동기화됩니다

4. Companion Server 및 자체 호스팅 데이터

TTDeck은 사용자가 설정한 Companion Server에 의존합니다. 해당 서버는 TeslaMate, Postgres, MQTT, Tesla 서비스, DNS 제공업체 및 사용자가 선택한 호스팅 서비스를 읽을 수 있습니다. 개발자가 운영하는 호스팅 서비스로 별도 제공하고 명시하지 않는 한, 이러한 서비스는 당사가 관리하지 않습니다. 각 서비스 사용에는 해당 약관과 개인정보 처리방침이 적용됩니다.

5. 제3자

TTDeck은 App Store, iOS Keychain, 로컬 알림, MapKit 표시, 카메라 권한 프롬프트 및 저빈도 설정 동기화에 사용하는 iCloud Key-Value Storage 등 Apple 플랫폼 서비스를 사용합니다. Apple이 App Store 또는 iOS를 통해 독립적으로 수집하는 데이터에 대해 당사는 책임지지 않습니다. 이 정책이 GitHub Pages에서 호스팅되는 경우 GitHub은 보안 및 사이트 운영을 위해 IP 주소 등의 정보를 처리할 수 있습니다.

6. 추적, 광고 및 데이터 판매

TTDeck은 사용자 데이터를 판매하지 않습니다. 현재 앱은 제3자 광고 SDK, 앱 간 추적 또는 제3자 분석 SDK를 사용하지 않습니다. 향후 분석, 충돌 보고, 원격 푸시 알림, 호스팅 Companion 서비스 또는 광고를 추가하는 경우 해당 버전을 출시하기 전에 본 정책과 App Store 개인정보 정보를 업데이트합니다.

7. 보관 및 삭제

로컬 앱 설정은 사용자가 변경하거나 바인딩을 재설정하거나 앱을 삭제할 때까지 기기에 남습니다. 앱 내 재설정은 이 기기의 읽기 전용 토큰과 기기 비밀값을 삭제합니다. 서버 측 기기 접근 취소, TeslaMate 데이터 삭제, Companion Server 로그 삭제 및 호스팅 인프라 데이터 삭제는 사용자가 관리하는 서버 또는 서비스에서 수행해야 합니다.

지원 정보는 요청 응답, 기록 유지 또는 법적 의무 이행에 합리적으로 필요한 기간 동안 보관됩니다. 삭제 요청은 xxxkkk0101@foxmail.com로 보내 주세요.

8. 사용자의 선택

  • Companion Server에 연결하지 않고 데모 모드를 사용합니다.
  • 앱에서 Companion Server URL 및 토큰을 변경하거나 제거합니다.
  • 개인정보 표시 옵션으로 정확한 위치, 경로 지도, VIN 접미사, 장소 라벨, 기기 토큰 해시를 화면에서 숨깁니다.
  • TTDeck 또는 iOS 설정에서 로컬 알림을 끕니다.
  • TTDeck 설정에서 이 기기의 바인딩을 재설정합니다.
  • Companion Server가 지원하는 경우 서버 측에서 기기 접근을 취소합니다.

9. 아동

TTDeck은 아동을 대상으로 하지 않으며 아동이 차량 또는 계정 데이터를 모니터링하는 데 사용해서는 안 됩니다.

10. 보안

TTDeck은 읽기 전용 토큰과 기기 비밀값을 iOS Keychain에 저장합니다. Companion Server, TeslaMate, 데이터베이스, 설정 비밀값, 호스팅 계정 및 네트워크 설정 보호는 사용자의 책임입니다. 공개 로그나 지원 채널에서 원본 토큰, 설정 비밀값, Tesla token, 데이터베이스 비밀번호, VIN, 정확한 집 좌표 또는 전체 경로 데이터를 공유하지 마세요.

11. 국제 사용

Companion Server 및 관련 서비스를 사용자와 사용 지역에 적용되는 법률에 따라 운영할 책임은 사용자에게 있습니다.

12. 변경

앱, 데이터 관행 또는 법적 요구사항이 변경되면 본 정책을 업데이트할 수 있습니다. 업데이트된 정책은 새 시행일과 함께 이 URL에 게시됩니다.
