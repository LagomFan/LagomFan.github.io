Informativa sulla privacy di TTDeck

Data di entrata in vigore: 2026-06-30

TTDeck è un'app di monitoraggio veicoli in sola lettura. Non è un prodotto ufficiale Tesla, non controlla il veicolo ed è progettata per collegarsi a un Companion Server configurato dall'utente.

1. Chi siamo

TTDeck è fornita da Fan Chentao. Puoi contattarci all'indirizzo xxxkkk0101@foxmail.com.

2. Riepilogo

La versione attuale di TTDeck non include SDK pubblicitari, SDK di analisi di terze parti o tracciamento tra app. L'app salva le impostazioni di connessione sul dispositivo e usa un token in sola lettura per richiedere riepiloghi del veicolo al Companion Server configurato dall'utente. I dati del veicolo sono destinati a rimanere in TeslaMate, nel Companion Server, nell'iPhone e nell'infrastruttura scelta e gestita dall'utente.

4. Companion Server e dati self-hosted

TTDeck dipende da un Companion Server configurato dall'utente. Tale server può leggere TeslaMate, Postgres, MQTT, servizi Tesla, provider DNS e hosting scelti dall'utente. Non controlliamo tali servizi salvo che forniamo e identifichiamo separatamente un servizio hosted gestito dallo sviluppatore. L'uso di tali servizi è regolato dai rispettivi termini e informative.

5. Terze parti

TTDeck usa servizi della piattaforma Apple come App Store, iOS Keychain, notifiche locali, visualizzazione MapKit, richieste di autorizzazione fotocamera e iCloud Key-Value Storage per la sincronizzazione impostazioni a bassa frequenza. Non siamo responsabili dei dati raccolti autonomamente da Apple tramite App Store o iOS. Se questa informativa è ospitata su GitHub Pages, GitHub può trattare informazioni dei visitatori, come indirizzi IP, per sicurezza e funzionamento del sito.

6. Tracciamento, pubblicità e vendita di dati

TTDeck non vende dati degli utenti. L'app attuale non usa SDK pubblicitari di terze parti, tracciamento tra app o SDK di analisi di terze parti. Se in futuro aggiungeremo analisi, crash reporting, notifiche push remote, un servizio Companion hosted o pubblicità, aggiorneremo questa informativa e le informazioni privacy di App Store prima del rilascio.

7. Conservazione ed eliminazione

Le impostazioni locali restano sul dispositivo finché non vengono modificate, l'associazione non viene reimpostata o l'app non viene eliminata. Il reset nell'app elimina da questo dispositivo il token in sola lettura e il segreto del dispositivo. Revoca lato server, eliminazione dati TeslaMate, log del Companion Server e dati dell'infrastruttura hosted devono essere gestiti sul server o servizio controllato dall'utente.

I messaggi di supporto sono conservati solo per il tempo ragionevolmente necessario a rispondere, mantenere registri o rispettare obblighi legali. Per richiedere l'eliminazione, contatta xxxkkk0101@foxmail.com.

8. Le tue scelte

  • Usare la modalità demo invece di collegare un Companion Server.
  • Cambiare o rimuovere nell'app URL e token del Companion Server.
  • Usare le opzioni privacy per nascondere posizione esatta, mappe percorso, suffisso VIN, etichette luoghi e hash token dispositivo.
  • Disattivare le notifiche locali in TTDeck o nelle Impostazioni iOS.
  • Reimpostare l'associazione di questo dispositivo dalle Impostazioni di TTDeck.
  • Revocare l'accesso del dispositivo dal Companion Server, se supportato.

9. Minori

TTDeck non è destinata ai minori e non deve essere usata da minori per monitorare dati del veicolo o dell'account.

10. Sicurezza

TTDeck usa iOS Keychain per token in sola lettura e segreto del dispositivo. Sei responsabile della protezione di Companion Server, TeslaMate, database, segreto di configurazione, account hosting e rete. Non condividere token grezzi, segreti, token Tesla, password database, VIN, coordinate esatte di casa o percorsi completi in log pubblici o canali di supporto.

11. Uso internazionale

Sei responsabile dell'uso del Companion Server e dei servizi collegati in conformità alle leggi applicabili a te e alla tua posizione.

12. Modifiche

Possiamo aggiornare questa informativa quando cambiano app, pratiche sui dati o requisiti legali. La versione aggiornata sarà pubblicata a questo URL con una nuova data.
