TTDeck プライバシーポリシー

発効日：2026-06-30

TTDeck は読み取り専用の車両モニタリングアプリです。Tesla の公式製品ではなく、車両を制御せず、ユーザーが設定した Companion Server に接続するよう設計されています。

1. 提供者

TTDeck は Fan Chentao が提供します。お問い合わせは xxxkkk0101@foxmail.com までご連絡ください。

2. 概要

現在の TTDeck には広告 SDK、第三者分析 SDK、アプリ横断トラッキング SDK は含まれていません。アプリは接続設定を端末に保存し、読み取り専用トークンを使ってユーザーが設定した Companion Server から車両サマリーを取得します。車両データは、ユーザーの TeslaMate、Companion Server、iPhone、およびユーザーが選択して管理するインフラに留まることを意図しています。

3. アプリが処理するデータ

データ目的保存先または送信先
Companion Server URL、データソースモード、選択車両、通知設定、プライバシー表示設定アプリ設定、表示内容の選択、端末上の設定保持iOS のアプリストレージに保存します。iCloud 設定同期を有効にした場合、Apple iCloud Key-Value Storage を通じてユーザーの iCloud アカウントにも低頻度で同期されます
読み取り専用 Companion トークンとデバイスシークレットこの iPhone を Companion Server に認証するためiOS Keychain に保存し、ユーザーが設定した Companion Server URL にのみ送信
セットアップシークレットと一回限りのペアリングトークンCompanion Server とのデバイス連携を開始または完了するためユーザーが入力し、設定した Companion Server に送信されます。TTDeck はセットアップ後に意図的に保存しません。
車両状態、バッテリー、充電、安全、走行履歴、ルート、位置、テレメトリ、診断サマリー読み取り専用画面、地図、履歴、ローカル通知チェックを表示するため設定済み Companion Server から取得し、端末上に表示します。開発者運営のクラウドに送信する設計ではありません。

ユーザーが任意で送信するサポートメール、スクリーンショット、ログ、診断情報サポート対応と問題調査のためユーザーが提供した場合のみ送信されます。送信前にトークン、正確な自宅位置、VIN、その他の機密情報を削除してください。

4. Companion Server とセルフホストデータ

TTDeck はユーザーが設定する Companion Server に依存します。そのサーバーは TeslaMate、Postgres、MQTT、Tesla サービス、DNS 事業者、ホスティング事業者などを読み取る場合があります。開発者運営のホストサービスとして明示しない限り、それらのサービスを当社は管理しません。各サービスの利用にはそれぞれの規約とプライバシーポリシーが適用されます。

5. 第三者

TTDeck は App Store、iOS Keychain、ローカル通知、MapKit 表示、カメラ権限プロンプト、低頻度の設定同期に使う iCloud Key-Value Storage など Apple のプラットフォームサービスを使用します。Apple が App Store や iOS を通じて独自に収集するデータについて当社は責任を負いません。本ポリシーを GitHub Pages でホストする場合、GitHub はセキュリティとサイト運用のため IP アドレス等を処理することがあります。

6. トラッキング、広告、データ販売

TTDeck はユーザーデータを販売しません。現在のアプリは第三者広告 SDK、アプリ横断トラッキング、第三者分析 SDK を使用していません。将来、分析、クラッシュレポート、リモートプッシュ通知、ホスト型 Companion サービス、広告を追加する場合、そのバージョンの公開前に本ポリシーと App Store のプライバシー情報を更新します。

7. 保持と削除

ローカル設定は、変更、連携リセット、またはアプリ削除まで端末上に保持されます。アプリ内のリセット操作は、この端末の読み取り専用トークンとデバイスシークレットを削除します。サーバー側のデバイスアクセス取消、TeslaMate データ削除、Companion Server ログ削除、ホスティング基盤データ削除は、ユーザーが管理するサーバーまたはサービス側で行う必要があります。

当社に送信されたサポート情報は、対応、記録保持、法的義務のため合理的に必要な期間保持されます。削除依頼は xxxkkk0101@foxmail.com までご連絡ください。

8. ユーザーの選択

  • Companion Server に接続せずデモモードを使用する。
  • アプリ内で Companion Server URL とトークンを変更または削除する。
  • プライバシー表示設定で正確な位置、ルート地図、VIN 末尾、場所ラベル、デバイストークンハッシュを非表示にする。
  • TTDeck または iOS 設定でローカル通知を無効にする。
  • TTDeck 設定からこの端末の連携をリセットする。
  • Companion Server が対応している場合、サーバー側でデバイスアクセスを取り消す。

9. 子ども

TTDeck は子どもを対象としておらず、子どもが車両またはアカウントデータを監視する目的で使用すべきではありません。

10. セキュリティ

TTDeck は読み取り専用トークンとデバイスシークレットを iOS Keychain に保存します。Companion Server、TeslaMate、データベース、セットアップシークレット、ホスティングアカウント、ネットワーク設定の保護はユーザーの責任です。公開ログやサポート窓口で、生トークン、設定シークレット、Tesla token、データベースパスワード、VIN、正確な自宅座標、完全なルートデータを共有しないでください。
